# 正则匹配



一般都是通用的知道基本概念和含义即可2

## 基本使用规则

### 普通字符

```
abc
```



### 特殊字符

```
. 匹配任意单个字符
* 匹配前一个字符0次或者多次
+ 匹配前一个字符1次或多次
```



## 案例库



### 匹配某一行含有某个字符串的行

```
.*字符串.*

.*flag.*

```



### 匹配同时含有多个字符串的行

```
.*data.*|.*information.*


```



### 匹配某字符串开头

```
^start.*
```



### 匹配以某字符串结尾

```
.*end$
```



### 匹配含有数字的行

```
.*[\d].*
```



### 匹配字母

```
.*[a-z].*
```



### 匹配倒数第五个字符为a的行

```
^.*(.{4}a)$
```



### 匹配换行符

```
\n
```



### 匹配所有大写字母

```
[A-Z]
```
