# 云计算

云是一种用于实现对可配置计算资源的共享池的普遍、方便、按需的网络访问的模型，该共享池可以以最小的管理工作或服务提供商交互快速地配置和释放。

云的五个基本特征

- 按需自助服务，用户可以在需要的时候自动配置资源
- 广泛网络访问，资源可以通过网络获得，并通过特殊机制访问
- 资源池化，汇聚存储和算例资源
- 快速弹性，云资源可以快速添加和删除，以匹配不断变化的需求
- 测量服务，监控、测量、报告资源使用量，为提供者和消费者提供资源使用的透明度，实现按资源用量付费的商业模式



部署在云中的IT资源成为服务，NIST区分了三种服务模型：

- SaaS软件即服务，云服务提供已经开发好的解决方案，随时可以使用，如gmail用于电子邮件
- PaaS平台即服务，为用户提供空间和基本工具来开始构建自己的定制解决方案，只需要关系编写代码和创建应用程序，不用管理操作系统
- IaaS基础设施即服务，提供虚拟服务器和存储，以便用户可以通过安装任何操作系统和软件来构建自己的IT环境。



NITS四种部署模型

- 公有云，商业模式
- 私有云，组织内部，本地托管数据
- 社区云，私有云的特殊情况，提供给多个组织，采用多租户模型
- 混合云，融合了公有云和私有云，允许数据和应用程序在它们之间移动



共享责任模型

云服务的安全和正常运行由云提供商和使用者共同负责

许多云漏洞都是由使用者的错误配置引入的。
